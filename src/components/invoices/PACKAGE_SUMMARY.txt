## 🚀 Quick Implementation (5 Steps)

### Step 1: Copy Files
```bash
cp -r components/invoices src/components/
```

### Step 2: Install Optional Dependency
```bash
npm install html2pdf.js
```

### Step 3: Import Component
```jsx
import InvoiceViewer from './components/invoices/InvoiceViewer'
```

### Step 4: Prepare Data
```jsx
const invoiceData = {
  invoiceNumber: 'INV-2026-00001',
  items: [...],
  // ... other fields
}
```

### Step 5: Render
```jsx
<InvoiceViewer invoiceData={invoiceData} />
```

## 📚 Code Examples

### Example 1: Basic Usage
```jsx
<InvoiceViewer invoiceData={{
  invoiceNumber: 'INV-2026-00001',
  customerName: 'John Doe',
  items: [...]
}} />
```

### Example 2: Calculate Totals
```jsx
import { calculateInvoiceTotals } from './invoice-utils'

const totals = calculateInvoiceTotals(items)
console.log(totals.grandTotal)  // ₹5000
```

### Example 3: Amount to Words
```jsx
import { amountToWords } from './invoice-utils'

const words = amountToWords(5432.50)
// "Five Thousand Four Hundred Thirty Two Rupees and Fifty Paise Only"
```

### Example 4: GST Calculation
```jsx
import { calculateGST } from './invoice-utils'

const { cgst, sgst, gstAmount } = calculateGST(1000, 5)
// cgst: 25, sgst: 25, gstAmount: 50
```

### Example 5: Data Transformation
```jsx
import { transformOrderToInvoice } from './invoice-utils'

const invoice = transformOrderToInvoice(orderData)
// Automatically formats order to invoice
```
